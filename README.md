# E-Commerce Sales Analysis

## Project Overview
This project analyzes e-commerce sales data using Excel, SQL, and Power BI. The objective is to clean raw data, calculate revenue after discounts, and generate business insights.

## Tools Used
- Microsoft Excel
- SQLite (SQL)
- Power BI

## Data Cleaning (Excel)
- Removed duplicate records
- Handled missing discount values
- Created final_amount column (amount - discount)

## SQL Analysis
- Total revenue after discount
- Revenue by city
- Top 3 customers by spending
- Revenue by product
- Total discount given

## Key Insights
- Laptop is the top-performing product by revenue
- Mumbai is the highest revenue-generating city
- Customer B is the highest spending customer
- Discounts significantly impact total revenue
- Laptop receives the highest discount share

## Power BI Dashboard
The dashboard includes:
- Total Revenue KPI
- Total Discount KPI
- Revenue by City
- Revenue by Product
- Top Customers
- Monthly Trend
- Interactive slicer (Product filter)

## Conclusion
This is an end-to-end data analysis project demonstrating data cleaning, SQL querying, and dashboard creation.
